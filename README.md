## scan2plot

Plots 2d lidar basd SLAM via matplotlib

![plot](https://github.com/Kucukcollu/scan_draw/blob/main/figures/plot.png)

## How to work ?
```bash
$ mkdir -p ~/catkin_ws/src
$ cd catkin_ws
$ wstool init
$ wstool merge ./src/scan2plot/depends.rosinstall
$ wstool update
```
